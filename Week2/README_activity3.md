# Week 2 - Activity 3: OOP Calculator

## Description
Rebuilt basic mathematical operations from Activity 1 using Object-Oriented Programming.

## OOP Concepts Used
1. **Class**: `Calculator` bundles data and operations together
2. **Constructor `__init__`**: Initializes object with `num1` and `num2` attributes 
3. **Methods**: `add()`, `subtract()`, `multiply()`, `divide()` operate on instance data
4. **Object Instantiation**: `calc = Calculator(num1, num2)` creates a Calculator object
5. **Encapsulation**: Data and methods are contained within the Calculator class

## Functions vs Methods
- **Functions**: `get_numbers()`, `display_results()`, `main()` exist outside the class
- **Methods**: Defined inside `Calculator` class and use `self` to access object data
