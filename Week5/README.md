## Description
The Clinic system is designed to manage or help patients to book appoinments, make payment and order there prescribed medications efficiently.

## Features
**Appointment Booking**
**Payment Processing**
**Order Medication**
**Patient Management**

## Workflow
1. Patient open clinic system
2. Patient register or log in
3. Patient view doctor and slots
4. Select doctor
5. Choose appointment
6. System checks the availability of the slot
7. Then Confirm booking
8. Patient makes payment
9. Booking confirmation is generated
10. Attend consultation
11. Medication is prescribed if required
12. Patient order Medication
13. Process order
14. Patient Logout
